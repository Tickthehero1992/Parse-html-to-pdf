1. Устанавливаешь Python 3 (brew install python3)
2. устанавливаешь pip3  (вроде он сразу встанет с питоном на маке)
3. в терминале "pip3 install -r requi.txt" терминал надо открыть в папке с проектом он установит модули для питона, может будет pip вместо pip3
4. кидаешь parser.py в папку
5. в терминале "python3 parser.py"
Создастся папка PDF_FILES в ней будут файлы конверченные(пример лежит в этой папке), названия будут на латинице
